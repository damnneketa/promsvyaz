ТЕРМОШКАФЫ

Термошкаф уличный ТШУ-2

Термошкаф уличный ТШУ-2 предназначен для установки в нём активного и пассивного телекоммуникационного оборудования, оборудования электропитания, аккумуляторных батарей. Термошкаф выполнен в антивандальном исполнении для установки вне помещения на «открытом воздухе».

Наружные размеры термошкафа ТШУ-2: - высота (без цоколя): 948 мм, - глубина (без кронштейна для крепления на стену): 475 мм, - ширина: 638 мм,

Термошкаф уличный ТШУ-6

Термошкаф уличный ТШУ-6-С предназначен для установки в нём активного и пассивного телекоммуникационного оборудования, оборудования электропитания. Термошкаф уличный ТШУ-6-А предназначен для установки аккумуляторных батарей. Термошкаф выполнен в антивандальном исполнении для установки вне помещения на «открытом воздухе».

Наружные размеры термошкафа ТШУ-6-С (ТШУ-6-А): - Высота (c цоколем): 1525 мм, - Глубина: 650 мм, - Ширина: 650 мм, - Высота цокольного отсека: 100 мм,
